# Privacy Policy — AI Factory (Personal Automation Tool)

This application is a personal tool built and operated by a single individual for
managing their own Etsy shop and Pinterest account. It is not a public service and
does not have external users.

**Data accessed:** This app connects to the Pinterest API using OAuth to create pins
on the developer's own Pinterest boards, linking to the developer's own Etsy listings.
It accesses only the data necessary to create and manage these pins (board information,
pin content).

**Data storage:** OAuth access tokens are stored securely in a private database
accessible only to the developer, solely to maintain the connection to the developer's
own Pinterest account. No data belonging to other Pinterest users is collected, stored,
or shared.

**Data sharing:** No data is sold, shared, or provided to any third party.

**Revoking access:** The developer can revoke this app's access at any time via
Pinterest's account settings.
