PROJECT NAME
AI Factory

MISSION

Build a fully autonomous AI company capable of creating, launching, managing, optimizing and expanding multiple online businesses with minimal human intervention.

The first business is Etsy.

The architecture MUST NOT be Etsy-specific.

Everything should be built as reusable infrastructure so future businesses (Shopify, YouTube, Amazon KDP, Gumroad, Creative Market, SaaS, etc.) can plug into the same system.

=========================================================
ARCHITECTURE PRINCIPLES
=========================================================

Everything must follow these rules.

Rule 1

Agents NEVER communicate directly.

Everything goes through the orchestrator.

Rule 2

Agents NEVER choose AI models.

Only AIRouter decides.

Rule 3

Agents NEVER know provider names.

Agents request capabilities.

Example:

generate_text()

reason()

generate_image()

vision()

embedding()

NOT

GPT

Claude

Gemini

Groq

etc.

Rule 4

Every task has a schema.

No free-form outputs.

Everything must validate.

Rule 5

Every AI response must be validated.

If validation fails:

Retry

Repair

Retry

Reject

Rule 6

Every major action must be logged.

Rule 7

Every important decision must be explainable.

=========================================================
KNOWN ISSUES ALREADY SOLVED
=========================================================

Never repeat these mistakes.

Do NOT let AI invent products.

Do NOT let Executor ignore Planner.

Do NOT allow markdown output.

Always enforce JSON.

Always strip markdown fences.

Always validate schema.

Always retry on invalid JSON.

Always repair malformed JSON.

Never trust model output.

OUTPUT STANDARD
=========================================================

No markdown.

No explanations.

No code fences.

No conversational text.
